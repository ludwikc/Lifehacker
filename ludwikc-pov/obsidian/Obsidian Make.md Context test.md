My goal is to have tags generated by Make.md in the *bulleted* form instead of a *comma-delimited list*.




Basic checklists:
- [ ] to-do
- [/] incomplete
- [x] done
- [-] canceled
- [>] forwarded
- [<] scheduling

Extras checklists:
- [?] question
- [!] important
- [*] star
- ["] quote
- [l] location
- [b] bookmark
- [x] information
- [x] savings
- [x] idea
- [x] pros
- [x] cons
- [f] fire
- [k] key
- [w] win
- [u] up
- [d] down
